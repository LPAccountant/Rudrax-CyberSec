3. Installation (Docker Mode) Recommended

Step 1: Clone the Repository

git clone https://github.com/ethics-learner/Rudrax-CyberSec.git
cd Rudrax-CyberSec

Step 2: One-Command Install

sudo bash deploy/install.sh docker

This will:

  • Install Docker and Docker Compose
  • Build all containers (Backend, Frontend, Ollama, Redis, Celery)
  • Pull default LLM model (llama3)
  • Configure Nginx reverse proxy with SSL
  • Set up systemd service for auto-start

Step 3: Access the Platform

Frontend:  https://your-domain.com
API Docs:  https://your-domain.com/api/docs
Admin:     lalittheonly@gmail.com / RudraX3#123nda

Docker Commands

docker-compose ps                    # Check status
docker-compose logs -f               # View logs
docker-compose down                  # Stop
docker-compose up -d --build         # Rebuild & restart
docker exec rudrax-ollama ollama pull mistral  # Pull new model

4. Installation (Bare Metal)

sudo bash deploy/install.sh bare

This mode installs Python, Node.js, and runs the backend directly with systemd. Useful for development or when Docker is not available.

7. AI Agents Reference

7.1 Voice Agent

Supports Hindi, English, and Hinglish (mixed). Auto-detects language from input text using regex patterns for Hindi characters and Hinglish keywords. Provides context-aware responses for cybersecurity, coding, networking, and general queries.

7.2 Browser & API Agent

  • Browse URL: Extract text, headers, links, or forms from any URL
  • Crawl Site: Spider up to N pages from a starting URL
  • API Tester: Send GET/POST/PUT/DELETE/PATCH requests with custom headers and auth
  • API Analyzer: AI-powered security analysis of API responses

7.3 OSINT Agent

One-click deep intelligence gathering that orchestrates WHOIS, DNS, subdomain enumeration, HTTP reconnaissance, email harvesting, and tech stack detection. Results are analyzed by AI and can be exported as HTML reports.

7.4 Multi-Agent Pipeline

Planner → Coder → Tester → Deployer pipeline with auto-agent detection based on message content. Agents include: Task Planner, Code Agent, Security Agent, Test Agent, Infra Agent, and Web Agent.

7.5 SOC Dashboard

Security Operations Center with:

  • Real-time event monitoring with severity classification
  • AI-powered event analysis
  • Auto-patch suggestion generation
  • Severity distribution charts
  • Timeline views and statistics

9. Troubleshooting

Ollama not connecting

docker exec rudrax-ollama ollama list    # Check models
docker logs rudrax-ollama                 # Check logs
docker restart rudrax-ollama              # Restart

Backend not starting

docker logs rudrax-backend               # Check errors
docker-compose down && docker-compose up -d  # Restart all

Redis connection issues

docker logs rudrax-redis                 # Check Redis logs
docker exec rudrax-redis redis-cli ping  # Test connection

SSL certificate issues

sudo certbot --nginx -d your-domain.com  # Re-run certbot
sudo certbot renew                        # Renew certificates
